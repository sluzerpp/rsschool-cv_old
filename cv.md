# Code Example
```JavaScript
function humanReadable(sec) {
  let h = Math.floor(sec / 3600),
      m = Math.floor(sec / 60 % 60),
      s = Math.floor(sec % 3600 % 60);
  if (h < 10) {
    h = "0"+h
  }
  if (m < 10) {
    m = "0"+m
  }
  if (s < 10) {
    s = "0"+s
  }
  return `${h}:${m}:${s}`
}
```
